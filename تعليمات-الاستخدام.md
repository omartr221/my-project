# تعليمات تشغيل نظام V POWER TUNING

## المشكلة الحالية
النظام لا يعمل على الشبكة المحلية من خلال Replit بسبب قيود الوصول الخارجي.

## الحل البديل: خادم محلي بسيط

### الخطوات:
1. **تحميل الملفات المطلوبة:**
   - `dist/` (المجلد الكامل)
   - `simple-server.js`
   - `package-simple.json`

2. **إعداد الخادم المحلي:**
   ```bash
   # انسخ الملفات إلى مجلد جديد
   mkdir vpower-local
   cd vpower-local
   
   # انسخ الملفات
   cp -r dist/ .
   cp simple-server.js .
   cp package-simple.json package.json
   
   # تثبيت المتطلبات
   npm install
   
   # تشغيل الخادم
   npm start
   ```

3. **الوصول للنظام:**
   - من نفس الجهاز: `http://localhost:3000`
   - من أجهزة أخرى: `http://192.168.1.102:3000`

### المتطلبات:
- Node.js مثبت على الجهاز
- الجهاز متصل بالشبكة المحلية
- البورت 3000 متاح

### استكشاف الأخطاء:
1. **إذا لم يعمل من الأجهزة الأخرى:**
   - تأكد من إيقاف الفايروول
   - تأكد من أن الجهاز في نفس الشبكة
   - جرب IP مختلف إذا تغير

2. **إذا ظهر خطأ "Application not found":**
   - تأكد من وجود مجلد `dist/public`
   - أعد بناء المشروع بـ `npm run build`

### مميزات الحل:
- ✅ يعمل بدون VPN
- ✅ أسرع من Replit
- ✅ لا يحتاج إنترنت للعمل
- ✅ يمكن تخصيصه حسب الحاجة
- ✅ يعمل على أي جهاز في الشبكة

## للمطورين:
إذا كنت تريد تطوير النظام محلياً:
1. انسخ المشروع كاملاً
2. تثبيت المتطلبات: `npm install`
3. تشغيل وضع التطوير: `npm run dev`
4. بناء الإنتاج: `npm run build`
