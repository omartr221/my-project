# دليل التشغيل النهائي - نظام V POWER TUNING

## المشكلة التي واجهتك:
المتصفح يُظهر "ERR_CONNECTION_REFUSED" لأن النظام لا يعمل على جهازك المحلي.

## الحل الكامل:

### الخطوة 1: تأكد من تثبيت Node.js
1. اذهب إلى https://nodejs.org
2. حمل النسخة LTS (الأحدث المستقرة)
3. ثبتها مع اختيار "Add to PATH"
4. أعد تشغيل الكمبيوتر

### الخطوة 2: تشغيل النظام
**استخدم الملف الجديد: `تشغيل-محلي-كامل.bat`**

هذا الملف سيفعل كل شيء تلقائياً:
- فحص Node.js
- تثبيت المكتبات (مرة واحدة فقط)
- تشغيل النظام على المنفذ 3000
- إرشادات واضحة

### الخطوة 3: فتح النظام
1. انتظر حتى تظهر "serving on port 3000"
2. افتح المتصفح
3. اذهب إلى: `http://localhost:3000`

### الخطوة 4: اختبار الوضع الأوفلاين
1. تأكد أن النظام يعمل ويظهر الواجهة
2. جرب إنشاء مهمة للتأكد أن كل شيء يعمل
3. افصل الإنترنت
4. النظام سيستمر في العمل!

## إذا لم يعمل:

### البديل الأول - Node.js Command Prompt:
1. ابحث في قائمة Start عن "Node.js command prompt"
2. افتحه
3. انتقل لمجلد المشروع:
   ```
   cd "C:\path\to\offline-system"
   ```
4. شغل:
   ```
   npm install
   npm run offline
   ```

### البديل الثاني - PowerShell كمدير:
1. اضغط Windows + X
2. اختر "Windows PowerShell (Admin)"
3. انتقل للمجلد:
   ```
   cd "C:\path\to\offline-system"
   ```
4. شغل:
   ```
   npm install
   npm run offline
   ```

## المنافذ المتاحة:
- المنفذ الافتراضي: 3000
- البدائل: 4000، 8000
