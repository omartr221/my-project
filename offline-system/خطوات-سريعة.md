# خطوات التشغيل السريعة

## الطريقة المضمونة:

### 1. استخدام Node.js Command Prompt
- ابحث في قائمة Start عن: **"Node.js command prompt"**
- افتحه (ستظهر نافذة سوداء)
- انتقل لمجلد المشروع:
  ```
  cd "C:\Downloads\offline-system"
  ```
  (غير المسار حسب مكان المجلد عندك)

### 2. ثبت المكتبات:
```
npm install
```

### 3. شغل النظام:
```
npm run dev
```

### 4. افتح المتصفح:
- اذهب إلى: **localhost:3000**

---

## إذا لم تجد "Node.js command prompt":

### خيار أ: إعادة تثبيت Node.js
1. اذهب إلى: https://nodejs.org
2. حمل النسخة LTS
3. ثبتها مع اختيار "Add to PATH"
4. أعد تشغيل الكمبيوتر

### خيار ب: استخدام المسار الكامل
```
"C:\Program Files\nodejs\npm.cmd" install
"C:\Program Files\nodejs\npm.cmd" run dev
```

---

## علامات النجاح:
- ظهور: "serving on port 3000"
- ظهور: "SQLite database initialized"
- بدون رسائل خطأ حمراء

## بعد النجاح:
- افتح أي متصفح
- اكتب: localhost:3000
- ستظهر واجهة النظام العربية
