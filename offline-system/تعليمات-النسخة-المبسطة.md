# النسخة المبسطة - بدون Node.js

## المشكلة:
Node.js غير معترف به أو غير مثبت صحيح في النظام.

## الحل البديل:
نسخة HTML مبسطة تعمل مباشرة في المتصفح بدون الحاجة لـ Node.js!

## ميزات النسخة المبسطة:

### ✅ تعمل بالكامل أوفلاين:
- لا تحتاج Node.js أو أي برامج إضافية
- تفتح مباشرة في أي متصفح
- تحفظ البيانات محلياً في المتصفح

### ✅ الميزات المتاحة:
- إنشاء مهام جديدة
- تتبع العمال الـ11 (غدير، يحيى، حسام، إلخ)
- تصنيف العمال (مهندس، فني، مساعد)
- تتبع الوقت وإنهاء المهام
- تاريخ المهام المكتملة
- إحصائيات فورية
- واجهة عربية كاملة مع شعار V POWER TUNING

## طريقة الاستخدام:

### الخطوة 1: فتح الملف
- اضغط مرتين على ملف `نسخة-بدون-nodejs.html`
- سيفتح في المتصفح الافتراضي

### الخطوة 2: الاستخدام العادي
- إنشاء مهام جديدة من النموذج
- اختيار الفريق (مهندس + فني + مساعد)
- تحديد نوع السيارة والوقت المقدر
- إنهاء المهام عند الانتهاء

### الخطوة 3: حفظ البيانات
- البيانات تُحفظ تلقائياً في المتصفح
- لا تحتاج قاعدة بيانات خارجية
- تبقى البيانات حتى بعد إغلاق المتصفح

## المقارنة مع النسخة الكاملة:

### النسخة المبسطة (HTML):
✅ تعمل فوراً بدون تثبيت  
✅ لا تحتاج Node.js  
✅ سهلة التشغيل  
❌ ميزات أقل  
❌ لا يوجد تزامن بين أجهزة متعددة  

### النسخة الكاملة (Node.js):
✅ ميزات كاملة  
✅ قاعدة بيانات SQLite  
✅ تزامن أفضل  
❌ تحتاج تثبيت Node.js  
❌ أعقد في الإعداد  

## للانتقال للنسخة الكاملة لاحقاً:
1. ثبت Node.js بشكل صحيح
2. أضف Node.js للـ PATH
3. أعد تشغيل الكمبيوتر
4. استخدم النسخة الكاملة

## تجربة النسخة المبسطة الآن:
افتح ملف `نسخة-بدون-nodejs.html` في المتصفح وجرب النظام!
