تعليمات تشغيل نظام توزيع المهام
=====================================

الآن Node.js يعمل صحيح! ✓

الخطوة الأخيرة:
--------------
1. اضغط مرتين على ملف "تشغيل-نهائي.bat"
2. انتظر رسالة "serving on port 3000"
3. افتح المتصفح واذهب إلى: http://localhost:3000

إذا ظهر خطأ "package.json غير موجود":
----------------------------------
- تأكد من تشغيل الملف من داخل مجلد offline-system
- اضغط كليك يمين على الملف واختر "تشغيل كمدير"

إذا ظهر خطأ "فشل في تثبيت المكتبات":
---------------------------------
- تأكد من الاتصال بالإنترنت
- جرب تشغيل Command Prompt كمدير

علامات النجاح:
--------------
✓ "serving on port 3000"
✓ "SQLite database initialized"
✓ لا توجد رسائل خطأ حمراء

عند النجاح:
-----------
- افتح المتصفح
- اذهب إلى: localhost:3000
- ستظهر واجهة النظام العربية

للمساعدة: تواصل معي إذا واجهت أي مشكلة
