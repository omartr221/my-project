# نظام توزيع المهام - النسخة المحلية

## تشغيل النظام بدون إنترنت

### متطلبات التشغيل:
- Node.js v18 أو أحدث
- npm

### خطوات التشغيل:

**الطريقة السهلة:**
1. شغل `check-system.bat` أولاً للتأكد من التثبيت
2. اضغط مرتين على ملف `simple-start.bat`
3. في المرة الأولى: سيثبت المكتبات تلقائياً (يحتاج إنترنت)
4. انتظر حتى تظهر رسالة "serving on port 3000"
5. افتح المتصفح على: `http://localhost:3000`

**إذا كان المنفذ 3000 مشغولاً:**
- استخدم `start-port-4000.bat` للمنفذ 4000
- أو `start-port-8000.bat` للمنفذ 8000

**التشغيل اليدوي:**
```bash
npm install
npm run dev
```

### ملاحظات هامة:
- النظام يحفظ البيانات في ملف tasks.db محلياً
- يعمل بدون الحاجة للإنترنت
- جميع الميزات متاحة: المهام، التوقيت، الأرشيف
- العمال المتاحون: غدير، يحيى، حسام، مصطفى، زياد، سليمان، علي، حسن، بدوي، عبد الحفيظ، محمد علي

### استكشاف الأخطاء:
- إذا لم يعمل النظام، تأكد من تثبيت Node.js
- إذا كان المنفذ 3000 مشغولاً، استخدم start-port-4000.bat أو start-port-8000.bat
- إذا ظهرت أخطاء في قاعدة البيانات، شغل fix-database.bat لإصلاحها
- تأكد من وجود ملف tasks.db في المجلد الجذر
