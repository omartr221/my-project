# دليل التحميل والاستخدام
## نظام توزيع المهام V POWER TUNING

---

## معلومات الإصدار
- **الإصدار**: نسخة أوفلاين كاملة
- **حجم الملف**: ~280 KB مضغوط
- **قاعدة البيانات**: SQLite محلية
- **عدد العمال**: 11 عامل جاهز

---

## خطوات التحميل والتثبيت

### 1. تحميل الملف
- حمل ملف `نظام-V-POWER-TUNING-أوفلاين.tar.gz`
- احفظه في مكان سهل الوصول (سطح المكتب مثلاً)

### 2. فك الضغط
**على Windows:**
- اضغط كليك يمين على الملف
- اختر "Extract All" أو استخدم برنامج مثل WinRAR/7-Zip
- اختر مجلد الاستخراج

**على Linux/Mac:**
```bash
tar -xzf نظام-V-POWER-TUNING-أوفلاين.tar.gz
```

### 3. متطلبات النظام
- **Node.js 18+** (تحميل من nodejs.org)
- **Windows 10+** أو **Linux** أو **macOS**
- **متصفح حديث** (Chrome, Firefox, Edge)

## طرق التشغيل

### الطريقة الأولى - Node.js Command Prompt (مضمونة):
1. ابحث في قائمة Start عن "Node.js command prompt"
2. افتحه وانتقل لمجلد المشروع:
   ```
   cd "C:\path\to\offline-system"
   ```
3. ثبت المكتبات:
   ```
   npm install
   ```
4. شغل النظام:
   ```
   npm run dev
   ```

### الطريقة الثانية - ملفات التشغيل التلقائي:
- اضغط مرتين على `تشغيل-نهائي.bat`
- إذا لم يعمل، جرب `تشغيل-بسيط.cmd`

### الطريقة الثالثة - PowerShell:
- اضغط مرتين على `تشغيل-بالباور-شيل.bat`

## ميزات النظام

### إدارة العمال:
- 11 عامل جاهز: غدير، يحيى، حسام، مصطفى، زياد، سليمان، علي، حسن، بدوي، عبد الحفيظ، محمد علي
- تصنيف العمال: فني، مساعد، مشرف، متدرب
- تتبع حالة العمال (متاح/مشغول)

### إدارة المهام:
- إنشاء مهام جديدة مع تحديد العمال لكل دور
- تتبع الوقت الفوري مع عدادات منفصلة
- إيقاف/استئناف المهام مع تسجيل الأسباب
- تحرير المهام والأوقات المقدرة

### نظام التسليم:
- تقييم ثلاثي (مقبول، جيد، ممتاز)
- حساب كفاءة العمل (الوقت الفعلي مقابل المقدر)
- أرشيف مع بحث وتصفية

### التقارير:
- تقارير تفصيلية قابلة للطباعة
- إحصائيات شاملة
- تصدير البيانات

## حل المشاكل

### مشكلة "Node.js not found":
- تأكد من تثبيت Node.js من nodejs.org
- أعد تشغيل الكمبيوتر بعد التثبيت
- استخدم "Node.js command prompt"

### مشكلة "Port already in use":
- شغل `npm run dev-alt` للمنفذ 4000
- أو أغلق البرامج التي تستخدم المنفذ 3000

### مشكلة تثبيت المكتبات:
- تأكد من الاتصال بالإنترنت (مطلوب للتثبيت الأول فقط)
- شغل Command Prompt كمدير
- استخدم `clean-install.bat` لتنظيف وإعادة التثبيت
