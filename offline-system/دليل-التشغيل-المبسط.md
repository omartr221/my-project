# دليل تشغيل النظام - حل المشاكل

## المشكلة الحالية
Node.js مثبت لكن غير معترف به في Command Prompt

## الحلول المتاحة

### الحل الأول: إعادة تشغيل الكمبيوتر
1. أعد تشغيل الكمبيوتر كاملاً
2. بعد إعادة التشغيل، جرب الملفات مرة أخرى

### الحل الثاني: استخدام PowerShell
1. اضغط Windows + R
2. اكتب: `powershell`
3. اضغط Enter
4. انتقل لمجلد النظام:
   ```
   cd "C:\path\to\offline-system"
   ```
5. شغل الأوامر:
   ```
   npm install
   npm run dev
   ```

### الحل الثالث: استخدام Node.js Command Prompt
1. ابحث في قائمة Start عن "Node.js command prompt"
2. افتحه
3. انتقل لمجلد النظام
4. شغل: `npm install` ثم `npm run dev`

### الحل الرابع: إعادة تثبيت Node.js
1. احذف Node.js من Control Panel
2. حمل النسخة الجديدة من nodejs.org
3. ثبتها مع اختيار "Add to PATH" 
4. أعد تشغيل الكمبيوتر

## تحقق من التثبيت
افتح Command Prompt واكتب:
- `node --version`
- `npm --version`

إذا ظهرت أرقام النسخ، فالتثبيت صحيح.

## معلومات إضافية
- النظام يعمل على المنفذ 3000
- العنوان: http://localhost:3000
- قاعدة البيانات: tasks.db (تُنشأ تلقائياً)
