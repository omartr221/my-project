# تعليمات تحميل وتشغيل نظام توزيع المهام
## V POWER TUNING - نسخة أوفلاين

---

## خطوات التحميل:

### 1. تحميل الملف المضغوط
- حمل ملف "نظام-توزيع-المهام-V-POWER-TUNING.zip"
- احفظه في أي مكان على جهازك

### 2. فك الضغط
- اضغط كليك يمين على الملف المضغوط
- اختر "Extract All" أو "استخراج الكل"
- اختر مكان الاستخراج (مثل سطح المكتب)

---

## خطوات التشغيل:

### الطريقة الأولى - Node.js Command Prompt (مضمونة):
1. **افتح قائمة Start**
2. **ابحث عن: "Node.js command prompt"**
3. **افتحه** (ستظهر نافذة سوداء)
4. **انتقل للمجلد:**
   ```
   cd "C:\Users\YourName\Desktop\offline-system"
   ```
   (غير المسار حسب مكان المجلد)
5. **ثبت المكتبات:**
   ```
   npm install
   ```
6. **شغل النظام:**
   ```
   npm run dev
   ```

### الطريقة الثانية - ملفات التشغيل التلقائي:
- اضغط مرتين على **"تشغيل-نهائي.bat"**
- إذا لم يعمل، جرب **"تشغيل-بسيط.cmd"**

---

## بعد التشغيل الناجح:

### علامات النجاح:
- ظهور: **"serving on port 3000"**
- ظهور: **"SQLite database initialized successfully"**
- بدون رسائل خطأ حمراء

### فتح النظام:
1. **افتح أي متصفح** (Chrome, Edge, Firefox)
2. **اكتب في شريط العنوان:** `localhost:3000`
3. **اضغط Enter**
4. **ستظهر واجهة النظام العربية**

---

## حل المشاكل:

### إذا ظهر "Node.js not found":
- Node.js غير مثبت
- حمله من: https://nodejs.org
- اختر النسخة LTS
- ثبته مع تفعيل "Add to PATH"
- أعد تشغيل الكمبيوتر

### إذا ظهر "Port already in use":
- شغل: `npm run dev-alt` بدلاً من `npm run dev`
- أو أغلق أي برامج تستخدم المنفذ 3000

### إذا فشل تثبيت المكتبات:
- تأكد من الاتصال بالإنترنت (مطلوب للتثبيت الأول فقط)
- شغل Command Prompt كمدير
