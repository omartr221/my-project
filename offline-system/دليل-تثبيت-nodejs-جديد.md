# دليل إعادة تثبيت Node.js والتشغيل النهائي

## بعد حذف Node.js - الخطوات الصحيحة:

### الخطوة 1: تحميل Node.js من جديد
1. **اذهب إلى الموقع الرسمي**: https://nodejs.org
2. **اختر النسخة LTS** (المربع الأخضر - الأكثر استقراراً)
3. **حمل النسخة Windows Installer** (.msi file)

### الخطوة 2: التثبيت الصحيح
1. **شغل الملف المحمل كمدير**:
   - اضغط كليك يمين على الملف
   - اختر "Run as administrator"

2. **أثناء التثبيت اختر هذه الخيارات**:
   ✅ Add to PATH environment variables
   ✅ Install npm package manager
   ✅ Install additional tools for Node.js

3. **أكمل التثبيت واتبع التعليمات**

### الخطوة 3: إعادة تشغيل الكمبيوتر
**مهم جداً**: أعد تشغيل الكمبيوتر بعد التثبيت لتحديث الـ PATH

### الخطوة 4: التحقق من التثبيت
1. **افتح Command Prompt جديد**:
   - اضغط Windows + R
   - اكتب: cmd
   - اضغط Enter

2. **جرب هذه الأوامر**:
   ```
   node --version
   npm --version
   ```

3. **يجب أن تظهر أرقام النسخ** (مثل v20.x.x)

### الخطوة 5: تشغيل النظام
1. **انتقل لمجلد offline-system**
2. **استخدم أحد هذه الملفات**:
   - `تشغيل-محلي-كامل.bat` (النسخة الكاملة)
   - أو `نسخة-بدون-nodejs.html` (النسخة المبسطة)

## إذا لم يعمل Node.js بعد التثبيت:

### الحل الأول - Node.js Command Prompt:
1. **ابحث في قائمة Start**:
   - اكتب: "Node.js command prompt"
   - افتحه إذا وُجد

2. **انتقل للمجلد**:
   ```
   cd "C:\path\to\offline-system"
   ```

3. **شغل النظام**:
   ```
   npm install
   npm run dev
   ```

### الحل الثاني - إضافة PATH يدوياً:
1. **اضغط Windows + R، اكتب**: `sysdm.cpl`
2. **اضغط "Environment Variables"**
3. **ابحث عن "Path" في System Variables**
4. **اضغط "Edit" ثم "New"**
5. **أضف**: `C:\Program Files\nodejs`
6. **اضغط OK واعد تشغيل الكمبيوتر**

## النسخة البديلة (إذا استمرت المشاكل):

### استخدم `نسخة-بدون-nodejs.html`:
- **تعمل فوراً** بدون Node.js
- **اضغط مرتين على الملف**
- **يفتح في المتصفح مباشرة**
- **جميع الميزات متوفرة**
