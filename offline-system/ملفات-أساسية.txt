الملفات الأساسية لنظام توزيع المهام
=====================================

هذه قائمة بالملفات الضرورية لتشغيل النظام:

## ملفات التشغيل (.bat/.cmd):
1. تشغيل-نهائي.bat - الملف الرئيسي للتشغيل
2. تشغيل-بسيط.cmd - نسخة مبسطة
3. clean-install.bat - تنظيف وإعادة تثبيت
4. check-system.bat - فحص النظام
5. find-nodejs.bat - البحث عن Node.js
6. تشغيل-بالباور-شيل.bat - تشغيل بـ PowerShell
7. start-with-powershell.ps1 - سكريبت PowerShell

## ملفات الإعداد:
1. package.json - إعدادات المشروع والمكتبات
2. tsconfig.json - إعدادات TypeScript
3. vite.config.ts - إعدادات Vite
4. tailwind.config.ts - إعدادات Tailwind CSS
5. postcss.config.js - إعدادات PostCSS
6. components.json - إعدادات المكونات

## ملفات الخادم (server/):
1. index.ts - نقطة بداية الخادم
2. routes.ts - مسارات API والـ WebSocket
3. db-sqlite.ts - اتصال قاعدة البيانات
4. storage-sqlite.ts - عمليات قاعدة البيانات
5. vite.ts - إعدادات Vite للخادم

## ملفات المخطط (shared/):
1. schema-sqlite.ts - مخطط قاعدة البيانات

## ملفات العميل (client/):
1. index.html - الصفحة الرئيسية
2. src/main.tsx - نقطة بداية React
3. src/App.tsx - المكون الرئيسي
4. src/pages/ - صفحات التطبيق
5. src/components/ - مكونات UI
6. src/lib/ - مكتبات مساعدة

## قاعدة البيانات:
1. tasks.db - قاعدة بيانات SQLite مع العمال الـ11

## ملفات التوثيق:
1. README.md - دليل التشغيل
2. تحميل-واستخدام.md - دليل شامل
3. حل-المشكلة.txt - حلول المشاكل
4. خطوات-سريعة.md - دليل مبسط

المجموع: حوالي 50 ملف
الحجم الإجمالي: ~2 MB (بدون node_modules)

هل تريد أن أبدأ بإرسال المحتوى ملف تلو الآخر؟
