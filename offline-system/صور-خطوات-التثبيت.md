# مكان خيار "Add to PATH" أثناء تثبيت Node.js

## الخطوات مع الصور الوصفية:

### الخطوة 1: بداية التثبيت
- شاشة الترحيب: اضغط "Next"

### الخطوة 2: اتفاقية الترخيص
- اقبل الاتفاقية: اضغط "I accept" ثم "Next"

### الخطوة 3: مجلد التثبيت
- المجلد الافتراضي: `C:\Program Files\nodejs`
- اتركه كما هو واضغط "Next"

### الخطوة 4: **هنا تجد خيار PATH** ⭐
**الشاشة المهمة**: "Custom Setup"

ستجد قائمة بالمكونات:
- ✅ Node.js runtime
- ✅ npm package manager  
- ✅ **Add to PATH** ← هذا هو المطلوب!
- ✅ Online documentation shortcuts

**تأكد أن جميع الخيارات محددة بعلامة ✅**

### الخطوة 5: أدوات إضافية
- خيار: "Automatically install the necessary tools"
- **اختر هذا أيضاً** (مفيد لبعض الحزم)

### الخطوة 6: التثبيت
- اضغط "Install"
- انتظر انتهاء التثبيت
- اضغط "Finish"

## إذا لم تجد خيار "Add to PATH":

### السبب المحتمل:
- قد يكون مُحدد تلقائياً في النسخ الحديثة
- أو قد يكون تحت اسم مختلف مثل:
  - "Add Node to PATH"
  - "Environment Variables"
  - "PATH Environment"

### البديل بعد التثبيت:
1. اضغط Windows + R
2. اكتب: `sysdm.cpl`
3. اضغط "Environment Variables"
4. ابحث عن "Path" في "System variables"
5. اضغط "Edit"
6. اضغط "New"
7. أضف: `C:\Program Files\nodejs`
8. أضف أيضاً: `C:\Program Files\nodejs\node_modules\npm\bin`
9. اضغط OK على جميع النوافذ
10. أعد تشغيل الكمبيوتر

## لاختبار التثبيت:
1. **بعد إعادة التشغيل**
2. **افتح Command Prompt جديد**
3. **اكتب**: `node --version`
4. **يجب أن تظهر**: `v20.x.x` أو نسخة مشابهة

## ملاحظة مهمة:
- **أعد تشغيل الكمبيوتر** بعد التثبيت
- **افتح Command Prompt جديد** للاختبار
- إذا لم يعمل، استخدم ملف `اختبار-nodejs-بعد-التثبيت.bat`

الخيار موجود في شاشة "Custom Setup" - تأكد من تحديد جميع الخيارات!
